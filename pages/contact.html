You can contact us in two different ways.

On GitHub (preferred)

Open a ticket to report a bug, or a discussion if you have any question, or if you want to ask for a new feature. This is where you’ll most probably get an answer.

Go to GitHub

On Framateam

If you don’t want to create an account on GitHub, you may be more comfortable to contact us via a chat hosted by a small French nonprofit organization.

Go to Framateam
